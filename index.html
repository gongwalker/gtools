Developer Toolbox

A powerful Chrome extension that bundles six commonly used developer tools: Timestamp Converter, Base Converter, Hash Calculator, Password Generator, Base64 Encoder/Decoder, and JSON Formatter.

  • Enter key shortcut supported

üî¢ Base Converter

  • Convert between binary, octal, decimal, and hexadecimal
  • Automatically filters invalid characters

  • Prompts when input contains invalid characters

üîê Hash Calculator

üìù Base64 Encode/Decode

üìã JSON Formatter

Installation

  1. Download or clone this project to your local machine
  2. Open Chrome and go to the Extensions page (chrome://extensions/)
  3. Enable "Developer mode"
  4. Click "Load unpacked"
  5. Select the project folder
  6. Done. Click the toolbar icon to start using it
